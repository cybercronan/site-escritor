# Ronan Cardoso | site de autor e professor

Site institucional criado para reunir minha trajetória, meus livros, cursos e formas de contato em um único endereço.

**Acesso:** [autor.ronancardoso.com](https://autor.ronancardoso.com/)

## Contexto

O projeto nasceu de uma necessidade do meu próprio trabalho como escritor e professor. A proposta foi organizar conteúdos de naturezas diferentes sem transformar a página em um catálogo excessivo. A leitura, a hierarquia das informações e o uso no celular orientaram as principais decisões.

## O que o site apresenta

* apresentação profissional e áreas de pesquisa
* livros publicados e links de compra
* cursos e projetos educacionais
* atuação no Instituto Mestra Natureza
* canais de contato

## Cuidados de implementação

* estrutura semântica e navegação por teclado
* link de salto para o conteúdo principal
* estados de foco visíveis
* imagens responsivas em WebP, com carregamento adequado a cada seção
* metadados para mecanismos de busca e compartilhamento em redes sociais
* layout adaptado a telas pequenas e grandes

## Tecnologias

HTML, CSS e JavaScript, sem etapa de build. A publicação usa GitHub Pages com domínio próprio.

## Executar localmente

```bash
python -m http.server 4173
```

Depois, abra `http://localhost:4173`.

## Status

Projeto publicado e em evolução conforme novos livros, cursos e atividades são incorporados.

Concepção, conteúdo e desenvolvimento por Ronan Cardoso. O código está disponível para consulta. Textos, imagens e identidade do site permanecem reservados aos respectivos autores.
